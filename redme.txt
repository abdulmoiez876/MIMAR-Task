1. For starting client follow these steps:
=> cd client
=> npm i
=> npm start
client will run on port 3000

2. for starting server
=> cd server
=> npm i
=> npm run dev
server will start running on port 8000

- When user logs in as customer, here he/she can see list of all products, on clicking picture on the image user will be navigated to products page (its just a basic page), user can add the products to cart and all the state management is done through redux
